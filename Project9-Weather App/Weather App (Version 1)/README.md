# Weather App (Version 1)

A simple command-line weather application written in Python.  
This project allows you to **get the current weather information** for any city using the [OpenWeatherMap API](https://openweathermap.org/api).

---

## Features

- Get current weather for any city in the world
- Shows temperature, weather description, humidity, and wind speed
- Simple and user-friendly command-line interface

---

## How It Works

- The user enters the name of a city.
- The program sends a request to the OpenWeatherMap API and retrieves the weather data.
- The weather information is displayed in the terminal.

---

## How to Use

1. **Clone the repository or copy the code into a file (e.g., `weather_app.py`).**
2. **Get a free API key from [OpenWeatherMap](https://openweathermap.org/api) and replace the value of `API_KEY` in the code.**
3. **Install the required dependency:**
    ```bash
    pip install requests
    ```
4. **Run the script:**
    ```bash
    python weather_app.py
    ```
5. **Enter the name of the city when prompted.**
6. **The current weather information will be displayed.**
