# To-Do App (Tkinter GUI)

A modern, user-friendly To-Do List application built with **Python** and **Tkinter**.  
This project allows you to manage your daily tasks with a beautiful, mobile-like interface and persistent storage.

## ğŸš€ Features

- **Add new tasks** with a simple input field
- **Delete tasks** with a single click
- **Edit tasks** and save changes
- **Mark tasks as done** (with a green checkmark and color)
- **Clear all tasks** with one button
- **Persistent storage**: tasks are saved in a JSON file (`tasks.json`)
- **Display total, done, and remaining tasks**
- **Modern and simple design** (fixed window size, colorful buttons, clean layout)
- **Responsive status bar**
